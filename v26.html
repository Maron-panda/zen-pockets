🔗 Koble til Google Sheets (valgfritt)

Dette gir automatisk backup til et Sheets-dokument hver gang du oppdaterer en status. Følg disse stegene:

1. Åpne et Google Sheets-dokument
2. Gå til Utvidelser → Apps Script
3. Lim inn koden under, klikk Publiser → Distribuer som nettapp, sett tilgang til «Alle»
4. Kopier URL-en inn i feltet under 

function doPost(e) {
  var d = JSON.parse(e.postData.contents);
  SpreadsheetApp.getActiveSpreadsheet()
    .getActiveSheet().getRange('A1')
    .setValue(JSON.stringify(d));
  return ContentService.createTextOutput('ok');
}

function doGet() {
  var v = SpreadsheetApp.getActiveSpreadsheet()
    .getActiveSheet().getRange('A1').getValue();
  return ContentService.createTextOutput(v || '{}')
    .setMimeType(ContentService.MimeType.JSON);
}

Merk: hvis synkroniseringen ikke fungerer pga. CORS, bruk Eksporter/Importer JSON i stedet.
